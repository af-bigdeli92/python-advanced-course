# Advanced Python Course

This repository contains the full teaching materials for my **Advanced Python Programming** course.  
It includes slides, code examples, exercises, and solutions designed for students who already understand Python basics and want to deepen their knowledge.

---

## Course Overview

This course covers advanced features of Python, focusing on writing clean, efficient, and professional-level Python code.  
By the end of this course, students will be able to:

- Write object-oriented code using advanced OOP concepts  
- Use iterators, generators, decorators, and context managers  
- Build and structure Python packages and modules  
- Handle errors and create custom exceptions  
- Work with files and manage Python environments  
- Understand concurrency (threading, multiprocessing)  
- Write asynchronous Python using `asyncio`  
- Apply common design patterns in Python
